# GBT2260

latest(201901) Chinese administrative divisions from [`MCA`](http://www.mca.gov.cn/article/sj/xzqh/2019/), and based on [`gb2260`](https://github.com/cn/GB2260.js)

## Install

```bash
npm i gbt2260
```

## Use

```js
const gbt2260 = require('gbt2260')
```

## Crawl Data

```bash
npm run generate
```
